Google Analytics

This application uses Google Analytics. It is used so that I can get an idea about what parts of the application is being used, and how much. It also helps give me external motivation for developing the application further.

There is not shared any information with them, that can identify you as a person. Location wise, it's only accurate down to municipality.

Where is my user data stored?

All of your user data, is locally stored on your computer. Whenever you download new auction house data or TSM api data, that too is stored on your computer in a database called indexedDB.

If you wish to move your settings from one device or browser to another, you can copy a export string from the settings section.

Using tools like CCleaner to clean your browser cache etc, will remove all your settings. So be aware of that.
