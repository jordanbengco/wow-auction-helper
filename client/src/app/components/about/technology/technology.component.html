The web client

The client is created using Angular v.5. My intention is to keep it at the latest release.

The application uses Angular Material combined with Bootstrap 4 for it's styling. Some custom modifications has been made also. For icons I've used Font Awesome.

Server side

The current backend is NodeJS and ExpressJS.

For the database that stores all the recipes, items and pets it uses MariaDB.

May I see the code?

All the code for the client is available on GitHub.

Both the front-end and back-end is available in the same repository.
