Items and pets All item and pet data are gathered from Blizzard's own data API and WoWhead. It is stored in my database as they do not support fetching more than one item at a time. This database is updated as new items are found in either recipes or at the auction house. Crafting recipes The crafting recipes are provided by WoWDB, and stored in my database. Missing recipes are dynamically added to the database as you and other users characters get new recipes. Statistical data (TSM) If you wish statistical data, like Market value, avg sale price, sale rate etc you need to register a TSM API key in the settings. Instructions on how to get a key, can be found there.
